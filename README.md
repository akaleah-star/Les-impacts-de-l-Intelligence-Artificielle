# Les-impacts-de-l-Intelligence-Artificielle
Rapport présentant les opportunités, risques et enjeux liés au développement de l’intelligence artificielle dans la société contemporaine.  Analyse des influences de l’IA sur le travail, l’éducation, la santé, la sécurité et l’environnement.  Étude critique de l’évolution de l’IA et de ses conséquences sur la vie humaine et les organisations.
